STEEL DRUM SAMPLES
==================

Put your .wav or .mp3 steel drum samples here with these exact names:

  steel-d4.wav   (or .mp3)  - D4 note
  steel-e4.wav   (or .mp3)  - E4 note
  steel-g4.wav   (or .mp3)  - G4 note  
  steel-a4.wav   (or .mp3)  - A4 note
  steel-c5.wav   (or .mp3)  - C5 note

WHERE TO GET SAMPLES:
- https://freewavesamples.com/instrument/steel-drums
- https://freesound.org (search "steel drum")
- https://sampleswap.org

Just download the WAV/MP3 files and rename them to match above.
